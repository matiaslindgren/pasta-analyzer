This is a search engine

  1. Copy paste your Python code
  2. Get code from GitHub that looks like your code

This search engine does not work in the conventional way of splitting the submitted query string into tokens that can be used to retrieve documents from the database. Instead, it parses the submitted code into an abstract syntax tree and uses the nodes of the syntax tree as tokens for retrieving similar syntax trees. User defined variable, function, class and function argument names are ignored.
