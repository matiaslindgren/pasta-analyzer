This is a code search engine

How do I use it?

  1. Copy paste your Python code
  2. Get code from GitHub repositories and the Python docs that looks like your code

How does it work?

The submitted code query is parsed into an abstract syntax tree and the nodes, i.e. subtrees, of the syntax tree are used as tokens for retrieving similar syntax trees from the index. User defined variable, function, function argument and class names are ignored. If you want to find specific functions or classes, regular search engines or the search functionality on GitHub will yield better results. The search results from this search engine are usually code that contain a structure that is similar to the submitted query.
